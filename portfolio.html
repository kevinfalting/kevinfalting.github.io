Wichitalks

A community driven speaking event based on the format of 20 slides auto-advancing every 15 seconds. Wichitalks was founded on January 1st, 2016 after reading Startup Communities by Brad Feld, and in response to the desire to connect many of the vibrant and passionate subcommunities within Wichita.

If you'd like to help support the website, you can find more information at Github.

Megaphone

A Rails idea sharing and discussion platform, similar to Hacker News. This application is being built with the help of Open Wichita in response to requests for something that could plug into current sites to act as a way to submit ideas.

If you'd like to help support the project, checkout the Github Project page.

Cristin Hancox Photography

A Jekyll website built for Cristin Hancox to support her photography. I utilized the Spectral Theme to help build the site.

If you'd like to see the code, checkout the Github page.

Pacerr

An iOS app built in Swift to help pace a run by generating a virtual pacer and providing feedback as to your position relative to the virtual pacer. Since it was made, new versions of Swift have been released and broken the code in this app. I intend to remake this app using Cordova so that it is easier to maintain and can be deployed to different mobile platforms.

If you'd like to see the code, checkout the Github page.
